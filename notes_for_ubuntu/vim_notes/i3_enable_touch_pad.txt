If your touchpad does not work after install i3wm

go to config file:
	/usr/share/X11/xorg.conf.d/40-libinput.conf

add the following code into conf file

	Section "InputClass"
		Identifier "libinput touchpad catchall"
		MatchIsTouchpad "on"
		MatchDevicePath "/dev/input/event*"
		Driver "libinput"
		Option "Tapping" "True"
	EndSection
